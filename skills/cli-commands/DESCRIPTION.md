# cli-commands

Skills in the cli-commands category:

- **code-review** — CLI 代码审查 — 审查当前 diff 的正确性 bug 和简化机会
- **simplify** — CLI 代码简化 — 审查变更代码的复用、简化、效率和抽象改进
- **verify** — CLI 验证 — 通过运行应用确认代码变更是否有效
- **run** — CLI 启动 — 启动并驱动项目应用查看变更效果
- **loop** — CLI 循环 — 按间隔重复执行 prompt 或 slash command
- **init** — CLI 初始化 — 创建新的 CLAUDE.md 文件记录代码库文档
- **security-review** — CLI 安全审查 — 完成当前分支待处理变更的安全审查
