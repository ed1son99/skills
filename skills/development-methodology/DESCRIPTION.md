# development-methodology

Skills in the development-methodology category:

- **test-driven-development** — 测试驱动开发 — 严格 red-green-refactor 循环
- **spec-driven-development** — 规格驱动开发 — 先写规格再编码
- **source-driven-development** — 源码驱动开发 — 基于官方文档做决策
- **doubt-driven-development** — 质疑驱动开发 — 每个非平凡决策经受对抗性审查
- **incremental-implementation** — 增量实现 — 逐步交付变更
- **interview-me** — 访谈式需求澄清 — 一问一答直到 95% 置信度
